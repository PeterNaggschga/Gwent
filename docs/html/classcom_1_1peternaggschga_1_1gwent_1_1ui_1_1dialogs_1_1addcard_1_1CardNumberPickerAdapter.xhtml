Detailed Description

A helper class responsible for initializing the NumberPicker views of an AddCardDialog (in #CardNumperPickerAdapter()) and adding UnitEntity objects with the selected attributes when addSelectedUnits() is called.

Constructor of a CardNumberPickerAdapter managing the NumberPicker views in the given ViewGroup using the given SquadManager.

Sets value bounds and NumberPicker.OnValueChangedListener for the pickers in the ViewGroup. The ViewGroup must be the ConstraintLayout with the id R.id::card_layout from popup_add_card.xml.
