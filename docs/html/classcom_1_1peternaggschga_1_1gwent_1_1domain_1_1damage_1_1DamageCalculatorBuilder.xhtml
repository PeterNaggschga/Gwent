Detailed Description

A builder class responsible for creating a decorator structure of DamageCalculator classes with the given parameters.

Should only be used by DamageCalculatorBuildDirector. Calls to #setWeather(), #setBond(), #setMoral() or #setHorn() should be in that exact order to create a correct DamageCalculator. Every function call is optional, i.e. every function should be called once or not at all.
