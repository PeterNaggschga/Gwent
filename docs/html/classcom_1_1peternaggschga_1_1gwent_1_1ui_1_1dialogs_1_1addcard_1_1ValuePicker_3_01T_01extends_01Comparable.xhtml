Detailed Description

A generic abstract wrapper class around NumberPicker used to select values of arbitrary types extending Comparable.

Must be subclassed to provide the displayed String values using #getDisplayString(). The default case, which is implemented in StringValuePicker, is that #displayIntegers maps the #selectableValues to a String resource id.
