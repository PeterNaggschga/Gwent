Detailed Description

A class managing the indicator views shown in the bottom bar of the IntroductionActivity.

The managed views show the user how many pages there are and which one they are on.

void updateIndicators ( @IntRange(from=0, to=4) int  position)
package

Updates the indicator views according to the given new position.

Sets the currentView to R.drawable#indicator_unselected and the view at the new position to R.drawable#indicator_selected. Saves the given position in currentView.

Parameters
positionInteger referencing the page represented by the managed indicators.
Exceptions
ArrayIndexOutOfBoundsExceptionWhen position is not in [0, 4].
