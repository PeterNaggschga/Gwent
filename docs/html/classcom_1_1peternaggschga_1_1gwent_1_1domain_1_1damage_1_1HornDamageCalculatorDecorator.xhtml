Detailed Description

A DamageCalculator class responsible for calculating the horn buff if necessary, i.e., if there are units with the com.peternaggschga.gwent.data.Ability::HORN ability that are not the unit the damage is calculated for or if there is a commander's horn in this row.

Acts as a ConcreteDecorator in the implemented decorator pattern and should decorate a WeatherDamageCalculator, a BondDamageCalculatorDecorator or a MoralDamageCalculatorDecorator.
