Detailed Description

A DamageCalculator class responsible for calculating the moral buff if necessary, i.e., if there are units with the com.peternaggschga.gwent.data.Ability::MORAL_BOOST ability that are not the unit the damage is calculated for.

Acts as a ConcreteDecorator in the implemented decorator pattern and should decorate a WeatherDamageCalculator or a BondDamageCalculatorDecorator for correct damage calculation.

Calculates the (de-)buffed damage of unit with the given id and the given base-damage.

Returns the given damage plus the number of moral boosts. If the unit itself has the com.peternaggschga.gwent.data.Ability::MORAL_BOOST ability, it is boosted one time less.
