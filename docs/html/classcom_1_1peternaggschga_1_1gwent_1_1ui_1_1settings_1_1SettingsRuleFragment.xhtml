Preference.OnPreferenceClickListener getSectionClickListener ( @NonNull RuleSection  section)
private

Creates an Preference.OnPreferenceClickListener that starts a new RuleActivity for the given RuleSection using an Intent.

The Intent provides the requested RuleSection to the RuleActivity using Intent#putExtra(String, Serializable) with RuleActivity#INTENT_EXTRA_TAG as a tag.
