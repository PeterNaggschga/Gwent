static Completable animatedSwitch ( @NonNull ImageView  view,
@DrawableRes int  resId 
)
static

Changes the resource of the given ImageView in an animation to the given resource id.

Animation is realized by creating a new ImageView on top of the given ImageView which is then faded out using the fade_out animation provided by Android. The operation is complete when the animation ends.

See also
android.R.anim::fade_out
