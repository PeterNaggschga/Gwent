Detailed Description

A class implementing SharedPreference.OnSharedPreferenceChangeListener for the THEME_PREFERENCE_KEY SharedPreference.

When this Preference is updated, the theme is switched in an animated way using ImageViewSwitchAnimator.

Only handles changes when the given key is equal to THEME_PREFERENCE_KEY. Updates the View objects in ballViews, cardViews, unitViews, and factionButton using ImageViewSwitchAnimator.
